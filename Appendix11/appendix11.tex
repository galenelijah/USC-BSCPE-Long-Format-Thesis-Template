\chapter{User Interface Screens}
\label{appendix:ui_screenshots}

This appendix provides a comprehensive visual catalog of the Graphical User Interface (GUI) designed for the USC Patient Information System (USC-PIS). While the core representative dashboards were discussed in Section 3.5, the following screen captures offer a complete visual walkthrough of the web application's role-based interfaces. These include the personalized student medical portals, the clinical staff management modules, and the advanced administrative configurations. The interface was engineered utilizing React 18 and Material-UI (MUI) components, ensuring progressive disclosure, responsive design across mobile and desktop environments, and a highly intuitive user experience for both the university student body and the USC Clinic staff.
